# OpenDrama (开源短剧) - 项目身份

## 🎭 项目名称
**OpenDrama (开源短剧)**

## 🎯 品牌定位
**开源·AI·短剧创作平台**
让每个人都能创作精彩短剧

## 🌟 名称含义
- **Open**: 开源精神，开放协作
- **Drama**: 短剧创作，影视内容
- **组合**: 开源的短剧创作平台

## 🎨 品牌标语
**开源短剧 · AI创作 · 人人皆是剧作家**

- **英文名**: OpenDrama

## 🎯 目标用户
- 📝 **内容创作者**: 短剧编剧、视频博主
- 🎬 **制作团队**: 小型影视制作公司
- 🤖 **AI开发者**: 技术爱好者、开源贡献者
- 📱 **个人用户**: 想要制作短剧的普通人

## 🛠️ 核心价值
1. **开放性**: 开源代码，透明可信
2. **智能化**: AI驱动，降低创作门槛
3. **一站式**: 从剧本到成片的完整工作流
4. **社区化**: 全球协作，共同进步

## 🏗️ 技术架构
- **后端**: Go + Gin + SQLite/PostgreSQL
- **前端**: Vue 3 + TypeScript + Element Plus
- **AI集成**: NewAPI统一接口 + TTS语音合成
- **部署**: Docker + 云服务支持

## 🎪 项目愿景
成为全球领先的开源AI短剧创作平台，让短剧创作像写作一样简单。
